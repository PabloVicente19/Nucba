¿Qué es javaScript?

javaScript es un lenguaje de programacion, en el cual se implementa para dar dinamismo e interactividad a un sitio web.

Javascript es un lenguaje de programación interpretado, por lo cual, puede ejecutarse en cualquier navegador. Los navegadores poseen un interprete de javascript de forma nativa.

¿Qué es un lenguaje de programación?

Un lenguaje de programación es un conjunto de normas gramaticales, que, hacen posible la comunicacion entre el ser humano y la computadora. Con un lenguaje de programación damos ordenes a un computador.

¿Cómo funciona Javascript?

Al cargar el sitio web, el navegador creara un lo que se conoce como DOM(Document Object Model). El DOM 'crea' un arbol de elementos html anidados.
